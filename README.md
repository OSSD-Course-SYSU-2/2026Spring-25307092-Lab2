# 💰 北北记账 (BeiBei Account Book)

基于 **HarmonyOS ArkTS** 开发的粉色系多功能记账应用，支持手机和平板多端自适应布局。

---

## 📱 功能详情

### 🏠 首页 — 记账

| 功能 | 说明 |
| :--- | :--- |
| 💵 收支记录 | 支持收入/支出两种模式，一键切换 |
| 📂 类别选择 | 支出 11 类：时尚、三餐、零食、酒水、生活、旅游、医疗、娱乐、交通、日常、人情；收入 5 类：职业收入、投资回报、退款、人情、其他收入；左右箭头切换 |
| 📅 日期选择 | DatePicker 自由选择记录日期，默认当天 |
| ✍️ 备注功能 | 每笔记录可添加备注说明 |
| 💰 余额计算 | 自动累加计算当前总余额，收入加、支出减 |
| 📋 历史记录 | 按时间倒序展示所有记录，显示类别、类型、金额、日期、备注 |
| 🗑️ 左滑删除 | 单条记录左滑出现红色删除按钮，余额自动修正 |
| 🧹 一键清空 | 清空所有记录，弹窗确认不可恢复 |
| 💾 数据持久化 | Preferences 存储，退出 App 自动保存，再次打开数据不丢失 |
| 🔄 自由流转 | 同华为账号下，手机和平板数据自动同步 |

### 📊 报表 — 统计分析

| 功能 | 说明 |
| :--- | :--- |
| 📅 月份切换 | 左右箭头切换月份，查看历史月度数据 |
| 📊 收支结余 | 收入/支出/结余三列卡片，点击收入或支出卡片切换下方分类详情 |
| 📈 分类排行 | 按金额从高到低排列，每个分类显示金额、百分比进度条（粉色系） |
| 📅 日历视图 | 点击日历按钮弹出月度日历弹窗，每一天按金额颜色标注 |
| 🎨 日历颜色 | 支出：无记录白色、<200 浅粉 #FDE0E1、200-1000 中粉 #F4A9AA、>1000 深粉 #E38A8B；收入：无记录白色、≤100 浅灰 #F5F5F5、>100 深灰 #D9DDE0 |
| 📉 日均统计 | 日历弹窗内显示当月日均支出或收入，切换收支自动更新 |
| 🔄 收支切换 | 支出/收入按钮切换日历显示内容 |

### 👤 我的 — 工具箱

| 功能 | 说明 |
| :--- | :--- |
| 👤 头像昵称 | 首字粉色圆形头像，点击可编辑昵称 |
| 🛒 购物清单 | 添加/删除待买物品，点击勾选标记已买（划线+变灰），数据持久化，退出不清空 |
| 💱 汇率换算 | 支持美元 USD、欧元 EUR、日元 JPY、港币 HKD、韩元 KRW、英镑 GBP 六种货币；点击切换货币自动更新汇率；支持人民币转外币 / 外币转人民币双向换算 |
| 💼 薪水计算 | 按广州市新税法计算：输入税前月薪，自动计算五险一金（养老 8%、医疗 2%、失业 0.2%、公积金 5%）、个人所得税（累进税率）、税后工资 |

### 📱📟 多端自适应

| 功能 | 说明 |
| :--- | :--- |
| 📱 手机布局 | 竖排单列布局，所有内容纵向排列 |
| 📟 平板布局 | 横排双列布局，首页余额卡片和收支按钮左右并排，类别和日期左右并排，输入框左右并排，历史记录双列网格 |
| 🔍 自动识别 | 窗口宽高比 > 0.85 且宽度 > 500 判定为平板，自动切换布局 |
| 📱📟 我的页面 | 手机端三个功能竖排三行，平板端横排三列 |
| 📌 底部导航 | 首页/报表/我的三页导航栏，选中粉色高亮，极浅灰背景 #FAFAFA |
