# 💰 北北记账 (Pink Account Book)

一个基于 **HarmonyOS ArkTS** 开发的粉色系记账应用。

## 📱 功能

- 💵 **收支记录**：支持收入和支出两种模式
- 📂 **类别选择**：支出 11 类（时尚、三餐、零食、酒水、生活、旅游、医疗、娱乐、交通、日常、人情）+ 收入 5 类（职业收入、投资回报、退款、人情、其他收入）
- 📅 **日期选择**：可自由选择记录日期
- ✍️ **备注功能**：每笔记录可添加备注说明
- 💰 **余额计算**：自动计算当前总余额
- 📊 **本月统计**：本月收入、支出、结余一目了然
- 📋 **历史记录**：所有记录列表展示，显示日期和备注
- 🗑️ **左滑删除**：单条记录可左滑删除，余额自动修正
- 🧹 **一键清空**：支持清空所有记录
- 💾 **数据持久化**：退出 App 自动保存，再次打开数据不丢失

## 🎨 界面

- 粉色系主题配色（#F4A9AA）
- 圆角卡片设计
- 浅粉色输入区域
- 简洁直观的交互

## 🛠️ 技术栈

- **开发工具**：DevEco Studio
- **语言**：ArkTS
- **平台**：HarmonyOS
- **数据存储**：Preferences

## 🚀 运行

1. 用 DevEco Studio 打开项目
2. 连接真机或启动模拟器
3. 点击运行按钮 ▶
