# 北北记账 (BeiBei Account Book)

基于 **HarmonyOS ArkTS** 开发的粉色系多功能记账应用，支持手机和平板多端自适应布局。

---

## 功能详情

### 首页 — 记账

**收支记录**
支持收入/支出两种模式，一键切换。收入金额自动加总，支出金额自动扣减。

**类别选择**
支出 11 类：时尚、三餐、零食、酒水、生活、旅游、医疗、娱乐、交通、日常、人情。收入 5 类：职业收入、投资回报、退款、人情、其他收入。左右箭头切换类别。

**日期选择**
DatePicker 自由选择记录日期，默认当天。选中项粉色高亮。

**备注功能**
每笔记录可添加备注说明，方便回顾消费详情。

**余额计算**
自动累加计算当前总余额，收入累加，支出扣减，实时更新。

**历史记录**
按时间倒序展示所有记录，显示类别、类型、金额、日期、备注。手机端单列，平板端双列网格。每条记录左滑出现红色删除按钮，删除后余额自动修正。

**一键清空**
清空所有记录，弹窗确认，不可恢复。

**数据持久化**
Preferences 本地存储，退出 App 自动保存，再次打开数据不丢失。



**自由流转**
同一华为账号下，手机和平板数据通过 AppStorage 自动同步。

---

### 报表 — 统计分析

**月份切换**
左右箭头切换月份，查看任意月度数据。

**收支结余**
收入、支出、结余三列卡片并排展示。点击收入卡片，下方显示收入各分类占比；点击支出卡片，下方显示支出各分类占比。选中的卡片有浅粉色背景高亮。



**分类排行**
每个分类按金额从高到低排列，显示类别名称、金额、百分比数字、粉色系进度条。进度条颜色各不相同，方便区分。

**日历视图**
点击日期旁边的日历按钮，弹出月度日历弹窗。每一天根据该日支出或收入金额显示不同颜色背景。

支出颜色规则：无记录显示白色，小于 200 元显示超浅粉色，200 到 1000 元显示中粉色，大于 1000 元显示深粉色。

收入颜色规则：无记录显示白色，100 元以内显示浅灰色，大于 100 元显示深灰色。

日历顶部有支出/收入切换按钮，点击切换显示内容。

**日均统计**
日历弹窗内显示当月日均支出或日均收入金额，切换收支时自动更新。

### 👤 我的 — 工具箱

**头像昵称**
粉色圆形头像显示昵称首字。点击编辑按钮可修改昵称。

**购物清单**
点击进入购物清单弹窗。输入物品名称添加待买物品。点击物品前的方框标记已买，已买物品文字变灰并加删除线。每条物品可单独删除。清空按钮一键清除所有物品。所有数据自动保存，退出不丢失。

**汇率换算**
支持六种常见货币：美元 USD、欧元 EUR、日元 JPY、港币 HKD、韩元 KRW、英镑 GBP。点击货币按钮切换，汇率自动更新。支持人民币转外币和外币转人民币两种方向自由切换。输入金额实时换算，结果显示对应货币金额。

**薪水计算**
按广州市新税法计算税后工资。输入税前月薪，自动计算五险一金（养老保险 8%、医疗保险 2%、失业保险 0.2%、公积金按最低档 5%）和个人所得税（按累进税率计算），显示税后实际到手工资。

### 📱📟 多端自适应

**自动识别**
通过窗口宽高比自动判断设备类型。宽高比大于 0.85 且宽度超过 500vp 判定为平板，自动切换平板布局。

**手机布局**
所有内容竖向单列排列。首页余额卡片居中，收支按钮横排，类别和日期上下排列，输入框上下排列，历史记录单列可滚动。我的页面三个功能入口竖排三行。

**平板布局**
首页余额卡片和收支按钮左右并排，类别和日期左右并排，金额和备注输入框左右并排，历史记录双列网格展示。我的页面三个功能入口横排三列。所有弹窗宽度自适应。

**底部导航**
首页、报表、我的三个页面底部导航栏切换。选中项粉色高亮，未选中灰色。背景极浅灰色。

## 🎨 界面设计

整体采用粉色系主题，主色调为温柔的粉色 #F4A9AA。深粉色 #E38A8B 用于支出相关元素，浅粉色 #FDE0E1 用于输入区域背景，页面背景为极浅粉色 #FFF5F5。

卡片使用白色背景、大圆角和柔和阴影，营造轻盈的层次感。导航栏背景为极浅灰 #FAFAFA，选中文字粉色，未选中灰色。

所有按钮采用大圆角设计，主按钮粉色背景白色文字。输入框浅粉色背景，深灰色文字。桌面图标为自定义粉色图标。

整体交互简洁直观，视觉统一柔和。

## 🛠️ 技术说明

开发工具使用 DevEco Studio，编程语言为 ArkTS，运行在 HarmonyOS 平台。

数据存储使用 Preferences，账单数据和购物清单都会自动持久化。跨设备流转通过 AppStorage 和 onContinue 实现。

多端自适应通过监听窗口尺寸变化，计算宽高比来判断手机或平板，动态切换布局。

日历组件使用 DatePicker 搭配自定义日历网格，图表使用 Progress 进度条组件模拟占比。

## 📁 项目结构

入口页面为 Index.ets，位于 entry/src/main/ets/pages/ 目录下，包含全部功能和三个页面的所有代码。

应用入口 EntryAbility.ets 位于 entry/src/main/ets/entryability/，包含自由流转的 onContinue 方法。

模块配置文件 module.json5 中 deviceTypes 包含 phone 和 tablet，支持多设备安装。

图标资源位于 entry/src/main/resources/base/media/，包含 startIcon 和 foreground 图标文件。字符串资源位于 element 目录下的 string.json。

---

## 🚀 运行指南

用 DevEco Studio 打开项目，连接真机或启动模拟器，点击顶部绿色三角形运行。

手机模拟器自动竖排布局，平板模拟器自动横排双列布局。支持 Previewer 切换 Phone 和 Tablet 预览。
