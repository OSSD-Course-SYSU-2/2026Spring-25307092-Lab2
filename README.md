# 💰 小粉记账 (BEIBEI Account Book)

一个基于 **HarmonyOS ArkTS** 开发的粉色系记账应用。

## 📱 功能

- 💵 **收支记录**：支持收入和支出两种模式
- 📂 **类别选择**：支出 11 类 + 收入 5 类，左右切换选择
- ✍️ **备注功能**：每笔记录可添加备注说明
- 💰 **余额计算**：自动计算当前余额
- 📋 **历史记录**：所有记录列表展示
- 🗑️ **左滑删除**：单条记录可左滑删除
- 🧹 **一键清空**：支持清空所有记录

## 🎨 界面

- 粉色系主题配色（#F4A9AA）
- 圆角卡片设计
- 简洁直观的交互

## 🛠️ 技术栈

| 项目 | 内容 |
|------|------|
| 开发工具 | DevEco Studio |
| 语言 | ArkTS |
| 平台 | HarmonyOS |

## 🚀 运行

1. 用 DevEco Studio 打开项目
