# 💰 北北记账 (BeiBei Account Book)

一个基于 **HarmonyOS ArkTS** 开发的粉色系记账应用，支持手机和平板多端自适应布局。

## 📱 功能

- 💵 **收支记录**：支持收入和支出两种模式
- 📂 **类别选择**：支出 11 类（时尚、三餐、零食、酒水、生活、旅游、医疗、娱乐、交通、日常、人情）+ 收入 5 类（职业收入、投资回报、退款、人情、其他收入），左右切换选择
- 📅 **日期选择**：可自由选择记录日期
- ✍️ **备注功能**：每笔记录可添加备注说明
- 💰 **余额计算**：自动计算当前总余额
- 📊 **本月统计**：本月收入、支出、结余一目了然，三列卡片展示
- 📋 **历史记录**：所有记录列表展示，显示类别、日期和备注
- 🗑️ **左滑删除**：单条记录可左滑删除，余额自动修正
- 🧹 **一键清空**：支持清空所有记录，带确认弹窗
- 💾 **数据持久化**：退出 App 自动保存，再次打开数据不丢失
- 📱📟 **多端自适应**：自动识别手机/平板，手机竖排布局，平板横排双列布局
- 🔄 **自由流转**：同一华为账号下，手机和平板数据自动同步

## 🎨 界面

- 粉色系主题配色（主色 #F4A9AA）
- 圆角卡片设计，柔和阴影
- 浅粉色输入区域（#FDE0E1）
- 收入/支出切换高亮
- 手机端历史记录居中对齐
- 简洁直观的交互

## 🛠️ 技术栈

- **开发工具**：DevEco Studio
- **语言**：ArkTS
- **平台**：HarmonyOS
- **数据存储**：Preferences（本地）+ AppStorage（跨设备流转）
- **自适应**：窗口宽高比判断断点

## 📁 项目结构
entry/src/main/ets/pages/Index.ets // 主页面（所有功能）
entry/src/main/ets/entryability/EntryAbility.ets // 应用入口（含自由流转）
entry/src/main/module.json5 // 模块配置（支持手机+平板）
entry/src/main/resources/base/element/ // 资源文件（颜色、字符串）

## 🚀 运行

1. 用 DevEco Studio 打开项目
2. 连接真机或启动模拟器
3. 点击运行按钮 ▶

### 多端预览
- 手机模拟器：自动竖排布局
- 平板模拟器：自动横排双列布局
- 支持 Previewer 切换设备预览

### 自由流转
- 手机和平板登录同一华为账号
- 数据自动同步，无缝切换
